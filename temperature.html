Temperature Dataset Information

Temperature dataset provides the seasonal and annual temperature recordings for the period of 58 years. This dataset offers the insights of climatic trends. Also, with this precise and seasonal temperature data of each year, helps in understanding long-term weather patterns.

Dataset Structure

Shape of Temperature Data set: (58, 6) i.e., it contains 6 attributes and 58 rows/items

Attributes Details

Column Name Attribute Type
YEAR Ordinal, Quantitative/Numerical attribute
ANNUAL, JAN-FEB, MAR-MAY, JUN-SEP, OCT-DEC Quantitative/Continuous Numerical attribute/Interval
