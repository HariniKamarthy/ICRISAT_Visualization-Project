Food Prices Dataset Information

This dataset consists of the periodical data of the different food prices in various cities across the country. Its a large collection of data including the prices in local currency and US dollars, making it to understand the food prices trends over time.

Dataset Structure

Shape of Food Prices Data set before data cleaning : (151810, 14). Shape of Food Prices Data set after data cleaning : (151284, 13).i.e., it contains 13 attributes and 151284 rows/items

Attributes Details

Column Name Attribute Type
date, year Ordinal, Quantitative/Numerical attribute
state, city, category, commodity, unit, priceflag, currency Nominal, Qualitative/Categorical attribute
latitude, longitude, price, usdprice Quantitative/Continuous Numerical attribute
